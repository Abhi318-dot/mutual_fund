# Mutual Fund Analytics Capstone Project

## Overview

This project was developed as part of the Bluestock Fintech Data Analyst Internship.

The objective is to analyze mutual fund performance, evaluate risk metrics, study investor behavior, and build an interactive Power BI dashboard for decision-making.

## Project Features

* Exploratory Data Analysis (EDA)
* Performance Analysis
* Risk Analysis (VaR & CVaR)
* Rolling Sharpe Ratio
* Cohort Analysis
* SIP Continuity Analysis
* Sector Concentration Analysis (HHI)
* Interactive Power BI Dashboard

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Power BI
* Jupyter Notebook

## Key Results

* 40 Mutual Funds Analyzed
* SIP Continuity Rate: 2.2%
* Majority of Funds Classified as Diversified
* Risk and Return Metrics Computed Successfully
